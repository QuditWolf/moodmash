## Hover States

### Card Hover
```

```

## Accessibility

### Focus States
- 2px outline
- Primary color
- 2px offset
- Visible on keyboard navigation

### Color Contrast
- Text on background: 4.5:1 minimum
- Muted text: 3:1 minimum
- Interactive elements: 3:1 minimum

### Touch Targets
- Minimum: 44x44px
- Sidebar items: 48px height
- Buttons: 40px minimum

## Design Principles

### Minimalism
- Plenty of whitespace
- Neutral colors
- Clean typography
- Subtle shadows

### Consistency
- Uniform spacing
- Consistent borders
- Predictable hover states
- Semantic colors

### Hierarchy
- Clear section titles
- Visual grouping
- Size variation
- Color contrast

### Responsiveness
- Mobile-first approach
- Fluid layouts
- Flexible grids
- Touch-friendly
